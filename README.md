# learncenter

### 功能描述
本项目采用spring boot + mybatis + thymeleaf模板引擎
* 实现了简单的登录、注册、退出功能
* 个人信息管理、教程信息管理模块
* 个人信息管理包括修改个人信息、修改密码功能
* 教程信息管理包括教程信息的添加、删除、修改、查询功能

### 开发环境
IDEA + JAVAa + MYSQL + MAVEN
